# EveryClass
Your school life, organized.

## 这是什么？
EveryClass 是一个课表软件，专为移动端设计。我们受到大屏课表软件 [ClassIsland](https://classisland.tech) 的启发，结合学校的“走班制”，决定开发 EveryClass。  
EveryClass 包含如下功能：

- [x] 课程管理和添加
- [x] 课程普通通知
- [ ] 课程实时通知
    - [x] iOS 实时活动 (灵动岛)
    - [x] Android 实时动态 (包括小米超级岛)
    - [ ] HarmonyOS 实况窗
    - [ ] 其他厂商
- [x] 多课表共存和切换
- [ ] 课程导入
    - [x] ClassIsland 档案
    - [ ] 其他课表软件
    - [ ] 教务系统
